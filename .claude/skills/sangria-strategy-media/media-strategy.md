# Media strategy - the conceptual thinking

> Strategy only. No plan line-items, pacing %, CPMs/CPAs, trafficking, reporting, analytics, or attribution - those belong to sangria-agency-os.

## Two-motor demand framework
- **Demand capture** - catch existing intent (search, retargeting, branded). Converts efficiently but capped by how much intent exists.
- **Demand creation** - build new intent (video, social, CTV, DOOH, podcast, creators, PR, LinkedIn/AI-visibility). Slower, but grows the pool capture later converts.
Capture without creation caps out; creation without capture leaks. Make creation *feed* capture.

## Funnel + channel-role architecture ("feeding the funnel")
Define the funnel and give **every channel one defined role with rules of engagement** - no redundancy, each fills a specific gap and states its contribution. Phase logic:
- **Awareness** - primary objective: build product/brand awareness across feeders. Role: reach + video views; grow the retargeting pool; lower downstream cost. (Heavy on dark video / ThruPlay, CTV to educate, social video + influencers.)
- **Consideration** - primary objective: qualified traffic to site. Role: revisit exposed users with RTBs, entice new prospects, fuel audience pools (retargeting + DemandGen + travel-intent data).
- **Performance** - primary objective: capture demand (generated by the campaign + existing on-site + in-market outside the brand). Role: convert at efficiency (e.g. Meta ASC against a custom conversion event; dynamic messaging).
State each phase's primary objective and KPI *type* (e.g. completed views; clicks/CTR; tickets/ROAS) - leave the actual numbers to the plan.

## Audience architecture (conceptual)
Build the audience plan in layers, not line items: **prospecting** (demographics + interests/psychographics) -> **contextual / lookalikes** (e.g. users similar to those who searched the category in the last 90 days) -> **retargeting** (searchers who didn't convert, microsite visitors who didn't convert) -> **already-impacted / owned** (past audiences, CRM, purchasers). Awareness audiences populate the pools that consideration/performance then harvest. Pair with owned/earned (influencers, organic social, email, owned banners, PR).

## Map media to the mindset journey
Align channels to **Dreaming -> Planning -> Sharing -> Booking -> Experiencing** and to **Attention -> Intent -> Influence**, so each placement matches where the traveler's head is. Weight channel selection by objective (represent "influence" by the size of the bet).

## B2B / LinkedIn creation (where relevant, e.g. Copa)
Treat LinkedIn as a demand-creation channel for professional audiences. Concepts: **Content Marketing Score** benchmarking vs a peer set; **whitespace 2x2** (engagement index x article volume) to find "engaging & uncommon" topics as the content opportunity; **business-traveler targeting** by combining professional attributes (function, seniority, industry) with behavioral/travel signals. (Use the thinking; leave the benchmark numbers out.)

## Core strategic principles
- **Runway changes dynamics** - a longer runway builds audiences and compounds vs. a sprint ("establish shorter flights across upper/mid funnel to add touch points").
- **"Retargeting is the engine; awareness is the fuel."** No fresh top-of-funnel -> retargeting caps out.
- **Sequence message and audience** - educate -> persuade -> close; cold -> engaged -> intent.
- **Big moments as announcement waves.**
- **Geo-phasing** - national -> feeder regions -> host market; don't spread funds too thin across first-timer feeders.
- **Let budget stay fluid** - shift in real time toward what's working.
- **Lock tracking before launch** - the setup is strategy (the readouts are not in this skill).

## Partner evolution (a maturation narrative)
Tell the channel story over time: pause/essentials in a downturn -> reintroduce proven publishers + test new ones (e.g. TripAdvisor) -> add social/TikTok using insights from Meta -> integrate PMax + DemandGen -> retain specialist partners (e.g. Crimtan) -> streamline to highest-impact + adopt new solutions (e.g. ASC). Frame "From the Now -> To the Next" per phase.

## The media-strategy deck arc (reusable)
1. **Recap / Foundation** - prior results as proof; channel winners; constraints; **key learnings turned into this cycle's strategic pillars / imperatives**.
2. **The Opportunity** - why this cycle is different; the new playbook / channel-role architecture; baseline & ambition.
3. **Expansion** - new-channel deep dives, each as **insight/stat hook -> strategic rationale -> recommended tactics -> funnel contribution -> sources**. (Channels: podcast, tiered creators/ambassadors, DOOH, data enrichment, CTV, AI-visibility/AEO/GEO, lead magnets, LinkedIn.)
4. **Integrated Activation Plan** - phased timeline; channel x funnel stack; **creative messaging by phase**; expansion/international logic.
5. **Budget philosophy, learning phase & projections** - below.

## Budget allocation philosophy (concept, not numbers)
Concentrate on **proven drivers**; fund **new channels with explicit kill criteria**; use **low-cash/high-upside** structures for creators (commission + perks); keep a small **always-on awareness layer** and a **test budget**; deploy progressively and pull back if efficiency degrades. (Phase weighting is a judgment - keep the percentages in the plan.)

## The "capital-control flywheel" (decision rule)
Scale spend **only while marginal efficiency stays below the economic ceiling** set by the offer's value; **hold** at the ceiling; **rebalance/tighten** when exceeded. Urgency near key dates lifts conversion and offsets some scaling loss. A decision rule, not a forecast.

## Learning phase (concept)
Early phases **build and validate the pipeline** (pool growth, lead quality, early signals) before being judged on conversion. Judge them on **pipeline, not final cost-per-acquisition**.

## Projections as scenarios (concept)
Model **Conservative / Target / Stretch** and state the **growth driver** behind each - ranges + logic, never guarantees.

## Partnership / co-funded logic
Partner, tourism-authority, and payment budgets run **under the host brand's strategy** (e.g. Stopover under Copa, co-funded by Promtur + ConnectMiles; destination pushes co-funded by tourism boards/payment partners). Align the partner's objective to a stage of the host funnel.
