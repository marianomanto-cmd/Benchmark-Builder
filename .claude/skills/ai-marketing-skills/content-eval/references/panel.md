# Content Eval Expert Panel

## Experts

### 1. Viral Hook Expert
**Lens:** Will this stop the scroll? Does the title create an irresistible curiosity gap?
**Scoring:** 90+ = can't NOT click. 80-89 = strong but seen similar. 70-79 = functional but forgettable. <70 = skip.
**Red flags:** Generic titles, no specificity, no tension, no number/data hook

### 2. Algorithm Expert
**Lens:** Will the platform recommend this? CTR, watch time, topic demand.
**Scoring:** 90+ = algorithm catnip. 80-89 = solid performer. 70-79 = niche appeal. <70 = dead on arrival.
**Red flags:** Low search volume topic, no rewatch/binge potential, too niche for broad recommendation

### 3. Founder Brand Strategist
**Lens:** Does this reinforce the brand's messaging pillars? Is it authentic to the founder's actual experience?
**Scoring:** 90+ = quintessentially on-brand, nobody else could make this. 80-89 = on-brand. 70-79 = tangential. <70 = off-brand.
**Red flags:** Generic advice anyone could give, no personal receipts, doesn't connect to core product/service

### 4. B2B Buyer Persona Expert
**Lens:** Would a CMO, VP Marketing, or CEO watch this and think "I need what they're selling"?
**Scoring:** 90+ = pipeline content. 80-89 = credibility builder. 70-79 = awareness only. <70 = wrong audience.
**Red flags:** Appeals to consumers not B2B, no buying signal, purely entertainment

### 5. Content Differentiation Expert
**Lens:** Is anyone else making this exact content? What's the unique angle?
**Scoring:** 90+ = nobody else can make this. 80-89 = differentiated angle. 70-79 = crowded topic, decent angle. <70 = commodity.
**Red flags:** 10+ creators already made this video, no unique data/experience, generic take

### 6. Short-form Adaptation Expert
**Lens:** Can this be clipped into 3+ viral Shorts? Does it have quotable one-liners?
**Scoring:** 90+ = 5+ clip moments built in. 80-89 = 3+ solid clips. 70-79 = 1-2 forced clips. <70 = unclippable.
**Red flags:** All framework/no story, no emotional peaks, too sequential to extract moments

### 7. Debate/Engagement Expert
**Lens:** Will this generate comments, shares, quote tweets, disagreements?
**Scoring:** 90+ = war in the comments. 80-89 = healthy debate. 70-79 = polite agreement. <70 = silence.
**Red flags:** Consensus opinion, no stakes, nothing to disagree with
